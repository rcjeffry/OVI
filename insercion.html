Método de ordenación de Inserción

La utilización del método de inserción consiste en insertar los datos de manera organizada de acuerdo a la posición que le corresponda por medio de la comparación de cada uno de los valores con todos los demás de forma organizada, al mismo tiempo se van desplazando de posición los valores comparados, es decir el primer dato es comparado con el segundo, si no se presenta el orden esperado se intercambian de posición, este mismo dato inicial se sigue comparando sucesivamente con todos hasta que se encuentre con aquel valor que cumple con el criterio, de esta manera es insertado el valor evaluado en la posición adecuada, se repite el proceso con el segundo dato hasta valorar todos las posiciones.


A continuación, se muestra el algoritmo para su debida implementación.


Entero[] insercion(entero v[], entero n)
	Entero: aux, i , j
	Para i = 2 hasta n hacer
		j = i – 1
		aux = v[i]
		Mientras v[j] > aux y j >= 1 hacer
			v[j+1] = v[j]
			j = j -1
		Fin mientras
		v[j+1] = aux
	Fin para
	Retornar v[]
Fin inserción
