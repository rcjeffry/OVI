Bienvenidos a este sitio web educativo, el cual tiene como tema principal la descripción de los métodos de ordenación utilizando programación orientada a objetos. Este material incluye la difusión de aquellos métodos directos utilizables en la mayoría de casos, para realizar operaciones de comparación con volumen de datos no complejos. La ordenación es una práctica indispensable a la hora de tener la información organizada, ya que refleja los datos de una manera comprensible para el usuario o algún programa específico que necesite realizar una operación detallada con estos valores. Existen diversos métodos que tienen como función realizar estas operaciones aritméticas, algunos de ellos pueden llegar a ser más eficientes que otros dependiendo de su efectividad y cantidad de comparaciones necesarias para completar el proceso, se dispone entonces en este sitio web de varios recursos educativos que pueden ser accedidos a través de los distintos links incluidos para los métodos de burbuja, inserción y selección.
